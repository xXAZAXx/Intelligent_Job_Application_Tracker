# Intelligent_Job_Application_Tracker

A simple and effective web application for managing job applications.
Users can track application status, store notes, and review progress
through a clean, responsive interface.

Built with Flask, Jinja2, HTML/CSS, and SQLite.

Note: Currently supports LinkedIn
